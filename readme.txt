[center][color=red][size=16pt][b]Block Censored Words[/b][/size][/color][/center]

This MODification gives you the option to block the post to be sent if a censored word is detected. SMF's default behaviour replaces "bad words" with your mask but allows the post to go through. With this, the post cannot be sent and an error message is issued.

     - English
